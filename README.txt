---------web_591自动化测试项目目录结构-------------
commom 存放公共模块
	-utils 存放公共模块，例如发送邮件、驱动封装、读取配置等
page 存放测试页面模块
config 存放设置信息，包括目录运行环境、用例执行环境等相关的配置参数
data 存放数据，例如驱动数据，图片等
report 存放测试报告
run 运行程序
testcase 存放测试用例的文件
logs 存放測試日誌、測試截圖等

----------預裝包---------------------------------
pip install -r requirements.txt
報告工具：allure-2.16.1
